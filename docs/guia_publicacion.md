# Guía de Publicación

## Pasos realizados

1. Se creó una ontología en Protégé utilizando OWL/RDF.
2. Se definieron clases, propiedades e individuos relacionados con ciberseguridad.
3. Se exportó la ontología en formato RDF/XML.
4. Se creó un repositorio público en GitHub.
5. Se integró Python con rdflib para realizar consultas SPARQL.
6. Se implementó un agente inteligente para recomendar controles.

---

## Publicación en GitHub

El repositorio fue publicado de manera pública para permitir acceso, colaboración y control de versiones.

---

## Tecnologías utilizadas

- Protégé
- OWL/RDF
- Python
- rdflib
